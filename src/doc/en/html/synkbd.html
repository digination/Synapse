Synapse Keyboard (SynKbd)

# of inputs: 1
# of outputs: 1
Resizable: No

Behaviour:

Synapse keyboard is a building block that allows to add key-typing functionality in workflows. It can be really useful eg to control an interactive shell in a synapp building block. To get The keyboard handle on a synkbd building block, you must click on the lower label , called "Acquire Keyboard" When a synkbd has the keyboard handle, its label becomes green. To release the keyboard, make another click on the green label,called "Keyboard Acquired"

Building Block Options:

Name

type: String
Default Value: Kbd#

Simply defines the blocks name. Also please note that a block name is not an UID ( meaning that you can have multiple blocks having the same name )

Color

type: HexString
Default Value: --

defines the building blocks color in the scheme
