Synapse Comment (Syncomment)

# of inputs: 1
# of outputs: 1
Resizable: Yes

Behaviour:

The syncomment building blocks are non-interactive, resizable labels which are used to put comments in a workflow, for instance to explain more prceisely the interractions between the blocks

Building Block Options:

Name

type: String
Default Value: Comment#

Simply defines the blocks name. Also please note that a block name is not an UID ( meaning that you can have multiple blocks having the same name )

Color

type: HexString
Default Value: #152233

defines the building blocks color in the scheme

Comment

type: String
Default Value: None

Defines the text that will be put in the comment label.
