Synapse Label (Synlabel)

# of inputs: 1
# of outputs: 1
Resizable: No

Behaviour:

This building block has a simple behaviour.

Building Block Options:

Name

type: String
Default Value: Label#

Simply defines the blocks name. Also please note that a block name is not an UID ( meaning that you can have multiple blocks having the same name )

Color

type: HexString
Default Value: #

defines the building blocks color in the scheme

Trailing CRLF

type: Boolean
Default Value: True

When this parameter is True, it automatically adds "\r\n" to the label string. It can be useful e.g when an application to which it is linkedneeds to validate the string on its STDIN
